## Getting Started

### Installation

1. Clone the repo
in ``Documents/Github``
   ```sh
   git clone https://github.com/ntijoh-my-lovell/villager-hunting-bingo
   ```
2. Install NPM packages
3. in ``villager-hunting-bingo/villager-bingo``
   ```sh
   npm install
   ```

### Running the application 
in ``villager-hunting-bingo/villager-bingo``
   ```sh
   ng serve
   ```
