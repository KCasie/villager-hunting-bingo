## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ntijoh-my-lovell/villager-hunting-bingo
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Running the application 
in ``villager-hunting-bingo/villager-bingo``
   ```sh
   ng serve
   ```
